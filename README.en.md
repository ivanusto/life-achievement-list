# Life Achievement System

🌐 **Languages / 多語言版本**: [繁體中文](README.md) | [English](README.en.md)

✨ **Online Demo / 線上體驗**: [https://ivanusto.github.io/life-achievement-list/](https://ivanusto.github.io/life-achievement-list/)

An interactive life milestone tracking journal that combines **achievement check-offs**, a **life domain footprint grid map**, and a **warm parchment-style scrapbook layout**. Designed for adventurers, creators, and everyone exploring the beautiful landscape of life!

---

## 🌟 Key Features

1. **Dashboard & Level System**: Real-time Level calculations (1 level for every 10 achievements checked) and customized cultivator titles (e.g., `🌱 Life Rookie`, `🎒 World Explorer`, `💼 Career Builder`, `🌟 Life Winner` to `🧙‍♂️ Legendary Life Master`).
2. **24 Life Domain Grid Map**: A responsive grid map allowing you to mark life domain nodes you have active footprints in (covering Health, Career, Hobbies, Wealth, and Relations).
3. **Canvas Radar Chart**: Built from scratch using native HTML5 Canvas (no external chart libraries) to plot your life exploration scores across 5 core dimensions.
4. **100 Milestones Database**: Spread across 10 categories (Learning & Growth, Work & Career, Wealth & Assets, Health & Sports, Love & Relations, Family & Kinship, Travel & Adventure, Art & Creation, Life & Hobbies, Ultimate Dreams), complete with hidden milestone reveals.
5. **10 Achievement Badges**: Unlock badges like `🎓 Academician`, `💼 Professional`, `💰 Capitalist`, `🏃 Athlete`, and `❤️ Lover` upon completing threshold goals.
6. **Shareable PNG Generation**: Instant generation of an 800x1200 high-res card displaying your nickname, avatar, level, badges wall, radar chart, and top 3 proudest achievements.
7. **Dual Cozy Themes**: Toggle between a dark "Cosmic Journal" (indigo starry theme) and a light "Warm Parchment" (cozy paper theme).
8. **Backup & Sync**: LocalStorage auto-saving with JSON import/export for cross-device manual synchronization.
9. **Zero Server Overhead**: 100% serverless, single static HTML page for maximum loading speed and absolute privacy.
10. **Custom Milestones**: Create up to 10 personalized accomplishments to check off and exhibit on your sharing card.

---

## 🛠️ Technical Stack

- **Core**: HTML5 / Vanilla CSS / Vanilla JavaScript (ES6)
- **Image Export**: HTML5 Canvas API (retina screen responsive)
- **Audio Effects**: Web Audio API (real-time chime synthesizer)
- **Animations**: CSS Keyframes + Canvas physical particle system (Confetti)
- **Storage**: LocalStorage API
- **Localization**: Built-in translation manager (i18n)
